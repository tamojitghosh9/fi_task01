actions :-

- data transform (checking for null/invalid values, changing data type where required, etc.)

- measures created within Sample-Superstore table:

i) Profit Margin = DIVIDE([Total Profit], [Total Sales], 0) 

ii) Total Profit = SUM('Sample - Superstore'[Profit])

iii) Total Sales = SUM('Sample - Superstore'[Sales])
